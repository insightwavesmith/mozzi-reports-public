1. 분석 개요

13개 키워드를 네이버에서 직접 검색하여 메인 노출 글 vs 종속 글의 차이를 분석했다. 각 키워드별로 검색 결과 상단 메인 카드에 배치된 게시물과 하단 2열에 종속 배치된 게시물을 비교하여 노출 결정 요인을 패턴으로 도출했다.

2. 발견한 5대 패턴

  • 1
    같은 카페 클러스터링
    같은 카페에서 같은 키워드 글 여러 개가 올라오면 → DIA 점수가 가장 높은 1개만 메인, 나머지는 전부 종속 배치.
    예시: "두피건선"으로 move79(해돌) 카페 글 3개 중 1개만 메인 → 나머지 2개는 클러스터링 피해.

    ⚠️ 같은 카페 = 자기끼리 경쟁 ✅ 다른 카페 분산 필수
  • 2
    RE(댓글) 종속은 무조건
    모든 키워드에서 RE로 시작하는 댓글 형식 게시물은 100% 종속 배치. 예외 없음.
    글의 내용·길이·이미지 유무와 무관하게 RE 타입 자체가 구조적으로 메인 노출 불가.

    원글 → 메인 노출 가능 RE 댓글 → 무조건 종속
  • 3
    메인 글의 공통점
    메인 노출 게시물에서 일관되게 관찰된 특성:
    • 본문 500자 이상 (짧은 글이 메인 배치된 사례 거의 없음)
    • 이미지 1장 이상 첨부
    • 경험담 형식 — "진단 받고~", "3개월 써봤는데~"
    • 댓글 3개 이상 달린 원글
    • 카페 C-Rank 높음 (맘카페, 대형 커뮤니티)
  • 4
    종속 글의 공통점
    종속 배치된 게시물에서 반복적으로 나타난 특성:
    • 같은 카페의 다른 글보다 DIA 점수 낮음
    • 본문 300자 미만
    • 이미지 없음
    • 댓글 0~1개
    • 제목이 단순 질문형 — "~괜찮을까요?", "~어때요?"
  • 5
    키워드 의도별 차이
    검색 의도에 따라 카페 노출 기회 자체가 다르다. 의도 분류가 먼저다.
    의도 예시 키워드 검색 결과 특징 카페 노출 기회
    의료/치료 건선, 아토피 닥터스픽 최상단 낮음
    후기/리뷰 교정후기, 라식후기 블로그 상단 보통
    고민/증상 가려움, 각질 카페 상단 높음
    쇼핑/추천 샴푸추천 쇼핑탭 우선 낮음
💡 핵심 인사이트: 카페 SEO 성패는 키워드 의도 선택 → 원글 타입 유지 → 카페 분산 → 콘텐츠 품질 순서로 결정된다.

3. 실제 사례 비교 — 두피건선

같은 "두피건선" 키워드 검색 결과에서 메인 노출된 글과 종속 배치된 글을 직접 비교했다.

✅ 메인
"두피건선 어떻게 하고 계세요?"
경험 공유형 제목, 상황 묘사 + 공감 유도. 방문자가 클릭하고 싶은 제목.
❌ 종속
"두피건선치료 괜찮을까요?"
단순 질문형 제목. 정보 가치 낮고 키워드 삽입이 어색. 클릭 유인 약함.
항목 메인 글 종속 글
제목 형식 경험 공유형 단순 질문형
본문 길이 길다 (500자+) 짧다 (300자 미만 추정)
이미지 ✅ 있음 ❌ 없음
댓글 수 3개+ 0~1개
게시 시점 선점 (먼저 게시) 후발
같은 카페 여부 ✅ ✅ (클러스터링 피해)

⚠️ 클러스터링 피해 구조

두 글이 같은 카페 소속임에도 종속 글이 하단 배치된 이유: 같은 카페에서 동일 키워드 글이 복수 존재하면 DIA 높은 1개만 메인 승격, 나머지는 자동 종속. 카페를 분산하지 않으면 자기 글끼리 자리를 빼앗는다.

📌 카페 선택 기준 요약

  • 1순위: 이미 해당 키워드로 메인 노출 이력 있는 카페
  • 2순위: 회원 10만+ 이상 대형 카페 (C-Rank 구조적 우위)

1 Motion AI 태깅 체계 상세 분석

Motion의 분류체계는 4개 카테고리로 모든 광고 소재를 태깅한다. 이 체계가 핵심이다. "어떤 유형이 잘 되는가"에 답하는 유일한 방법.

소재의 "생김새"를 분류한다. 영상/이미지가 어떤 방식으로 만들어졌는가.

1-2. Hook Tactic — 5대 훅 분류

$1M+/월 계정 데이터 분석에서 Motion이 추출한 고효율 훅 5가지. 광고 첫 3초의 유형을 분류한다.

Confession
잘못이나 의외의 사실을 솔직하게 인정하며 시작
🎯 "사실 나도 처음엔 안 믿었는데..."
분석: 첫 문장에 부정어/고백어 패턴 감지
Bold Claim
강하고 대담한 주장으로 즉시 주목을 끄는 방식
🎯 "이것만 있으면 다 됩니다"
분석: 첫 문장에 과장/절대 표현 ("유일한", "최고", "무조건") 감지
Relatability
타겟의 공감을 유도하며 "나 얘기 같다"는 느낌 형성
🎯 "혹시 이런 경험 있으세요?"
분석: 첫 문장에 2인칭 대명사 + 질문형 감지
Contrast
전/후, 좋음/나쁨 등 대비 구조로 변화를 강조
🎯 "전에는... 지금은..."
분석: 비교 접속사 ("하지만", "반면", "VS") + split screen 동시 감지
Curiosity
정보 격차를 활용해 "끝까지 봐야 알 수 있다"는 긴장감 형성
🎯 "이걸 모르면 큰일 납니다" / "아무도 안 알려주는..."
분석: 불완전 정보 패턴 + 의문문 감지 (비밀, 숨겨진, 알려지지 않은)
→ bscamp 적용 액션

영상 소재의 첫 3초 자막/텍스트 추출 → Gemini로 훅 유형 분류 → hook_tactic 태그 저장. 이후 "어떤 훅이 우리 수강생 계정에서 CTR 높은지" 비교 가능.

1-3. Messaging Angle — 8대 메시지 앵글

같은 제품이라도 어떤 "각도"로 말하는가. 카피 + 비주얼에서 자동 분류.

💰
Price / Offer
"50% 할인", "오늘만 이 가격"
✨
Quality / Premium
"100% 오가닉", "프리미엄 성분"
👥
Social Proof
"10만 명이 선택", "별점 4.9"
💝
Emotional
"당신의 삶이 바뀝니다"
🔧
Problem / Solution
"이 고민, 해결했습니다"
⏰
FOMO
"오늘만", "마감 임박", "한정수량"
🏆
Authority
"전문가가 추천", "MBC 방영"
🌿
Lifestyle
"이런 삶을 원한다면"
→ bscamp 적용 액션

광고 카피(primary_text, headline) 분석 → Gemini로 앵글 분류 → messaging_angle 저장. "우리 수강생들이 Price/Offer에만 몰려 있다" → "Social Proof 앵글 테스트 해보세요" 처방 자동화.

1-4. Intended Audience — 타겟 자동 추정

👤 AI가 소재만 보고 타겟을 추정한다

소재의 시각적 요소(등장인물 특성, 배경, 색감) + 카피 톤에서 이 광고가 "누구한테" 말하는지 자동 추정.

  • 예: "예산 걱정하는 초보 사업자" — 가격 강조 + 단순한 배경 + 직설적 카피
  • 예: "30대 스킨케어 관심 여성" — 여성 등장인물 + 클린 미학 + 성분 강조
  • 예: "퍼포먼스 마케터" — 숫자/지표 중심 + 대시보드 화면

핵심 활용: Meta adset의 실제 타겟팅 설정과 비교 → "소재 톤은 20대 겨냥인데 타겟팅은 40대로 되어 있음" → 불일치 경고 가능.

→ bscamp 적용 액션

소재 이미지의 인물 특성 + 카피 톤 분석 → intended_audience 자유형식 텍스트로 저장. 향후 adset 타겟팅 설정과 자동 비교하여 불일치 감지.

2 Comparative Reports 구현 방법

"UGC vs 모션그래픽 중 어느 쪽 ROAS가 높아?" 같은 질문에 데이터로 답하는 리포트. Motion의 킬러 기능. 지금 bscamp에 없다.

🟣 Motion 방식

Group by [AI 태그] → 각 그룹의 평균 Spend / ROAS / CTR 비교 → 가로 바 차트

  • Visual Format별: "Demo 4.2x ROAS" vs "Skit 2.1x ROAS"
  • Hook 유형별: "Bold Claim CTR 3.8%" vs "Relatability CTR 2.1%"
  • Messaging Angle별: "Social Proof ROAS 5.1x" vs "Price 3.2x"

bscamp 적용 설계

-- Visual Format별 성과 비교
SELECT
  cea.visual_format,
  AVG(dai.roas) AS avg_roas,
  AVG(dai.ctr) AS avg_ctr,
  SUM(dai.spend) AS total_spend,
  COUNT(DISTINCT cea.creative_id) AS ad_count
FROM creative_element_analysis cea
JOIN daily_ad_insights dai ON cea.creative_id = dai.creative_id
WHERE cea.account_id = $1
  AND dai.date >= NOW() - INTERVAL '30 days'
  AND cea.visual_format IS NOT NULL
GROUP BY cea.visual_format
ORDER BY avg_roas DESC;

-- Hook Tactic별 성과 비교
SELECT
  cea.hook_tactic,
  AVG(dai.ctr) AS avg_ctr,
  AVG(dai.roas) AS avg_roas,
  COUNT(*) AS ad_count
FROM creative_element_analysis cea
JOIN daily_ad_insights dai ON cea.creative_id = dai.creative_id
WHERE cea.hook_tactic IS NOT NULL
GROUP BY cea.hook_tactic
ORDER BY avg_ctr DESC;

UI 설계

  • 바 차트: 가로 바. X축 = ROAS/CTR, Y축 = 포맷/훅 유형. 색상으로 성과 등급 표시.
  • 카드형 리포트: 각 포맷별로 "평균 ROAS / 집행 건수 / 대표 소재 썸네일" 카드.
  • 포맷별 대표 소재: 해당 Visual Format에서 ROAS 1위 소재 썸네일 표시.
  • 필터: 기간 선택 (7일/30일/90일) + 태그 카테고리 선택 (Visual Format / Hook / Angle).
→ bscamp 적용 액션

1단계 태깅 완료 후, Supabase View로 집계 쿼리 등록 → 프론트 "비교 분석" 탭 추가. 수강생이 "어떤 유형이 내 계정에서 이기고 있는지" 바로 볼 수 있게.

3 Launch Analysis — 신규 소재 72시간 추적

새 소재를 올리면 초기 성과가 위너/루저를 결정한다. Motion은 이 "골든 타임"을 자동으로 추적한다.

🟣 Motion 방식

  • 새 광고 감지 — created_at 기준으로 신규 소재 자동 감지
  • 일별 성과 추적 — 첫 72시간(D+1, D+2, D+3)이 핵심 판단 시점
  • 자동 분류: Early Winner / Scaling / Declining / Watch
  • 벤치마크 대비 상위 20% 진입 → "이거 예산 올리세요" 알림
  • D+3까지 하위 30% → "이거 끄세요" 알림

🟢 bscamp 적용 설계

기존 인프라 활용: creative_media.first_served_at 컬럼 이미 존재. collect-daily에서 최초 집행일 기록 중.

  • collect-daily에서 first_served_at이 72시간 이내인 소재 → new_creative = true 플래그
  • D+1 / D+3 / D+7 성과 집계 뷰 추가 (집계 기준: ROAS, CTR, Spend)
  • 벤치마크 대비 자동 등급 판정: creative_grade 필드
  • 슬랙/카카오 알림: "새 소재 [이름]이 D+3 기준 상위 15%입니다. 예산 올리는 것 추천."
-- 신규 소재 D+1/3/7 성과 추적 뷰
SELECT
  cm.creative_id,
  cm.thumbnail_url,
  cm.first_served_at,
  DATE_DIFF('day', cm.first_served_at, NOW()) AS days_live,
  AVG(CASE WHEN dai.date <= cm.first_served_at + 1 THEN dai.roas END) AS d1_roas,
  AVG(CASE WHEN dai.date <= cm.first_served_at + 3 THEN dai.roas END) AS d3_roas,
  AVG(CASE WHEN dai.date <= cm.first_served_at + 7 THEN dai.roas END) AS d7_roas
FROM creative_media cm
JOIN daily_ad_insights dai ON cm.creative_id = dai.creative_id
WHERE cm.first_served_at >= NOW() - INTERVAL '14 days'
GROUP BY cm.creative_id, cm.thumbnail_url, cm.first_served_at;
→ bscamp 적용 액션

collect-daily 크론에 "신규 소재 감지 + 등급 판정" 로직 추가. 대시보드에 "신규 소재 추적" 카드 섹션 추가. 슬랙 알림 연동.

4 Inspo (경쟁사 분석) 강화

🟣 Motion Inspo 기능

  • Meta Ad Library 크롤링 → 브랜드별 광고 정리
  • Visual Format 20개 자동 분류 (모든 경쟁사 광고에 태깅)
  • Days Active (게재 기간) 표시 — 오래 살아있는 광고 = 효과 있는 광고
  • Media Mix 차트 — 경쟁사의 영상/이미지/캐러셀 비율
  • "Formats you haven't tried" — 내가 안 써본 포맷 중 경쟁사에서 잘 되는 것 추천

🟢 bscamp 현황 + 추가 필요 사항

현재 있는 것:

  • Ad Library 수집 파이프라인 — competitor_ads 테이블 존재
  • 브랜드별 광고 리스트 조회

추가 필요:

  • Visual Format 자동 분류 — 경쟁사 광고에도 L1 태깅 적용 (수강생 광고와 동일한 파이프라인)
  • Days Active — first_seen_at ~ last_seen_at 기간 계산 + 표시
  • Media Mix 차트 — 영상/이미지/캐러셀 비율 도넛 차트
  • "안 써본 포맷 추천" — 수강생 소재 Visual Format 목록 vs 경쟁사 포맷 목록 → 교집합의 여집합
-- "안 써본 포맷 추천" 쿼리
SELECT DISTINCT ca.visual_format
FROM competitor_ad_analysis ca
WHERE ca.competitor_brand_id = $brand_id
  AND ca.visual_format NOT IN (
    SELECT DISTINCT cea.visual_format
    FROM creative_element_analysis cea
    WHERE cea.account_id = $account_id
      AND cea.visual_format IS NOT NULL
  )
ORDER BY ca.avg_days_active DESC;
-- 결과: 경쟁사는 쓰는데 나는 안 써본 포맷 목록 (게재 기간 긴 순)
→ bscamp 적용 액션

경쟁사 광고 수집 시 Visual Format 자동 분류 + Days Active 계산 추가. "안 써본 포맷 추천" 기능으로 수강생 소재 다양화 가이드. 수강생 소재 획일화 문제 해결.

5 Andromeda 최적화 점수 — bscamp 독자 기능 제안

🤖 Andromeda = Meta의 광고 배포 AI

Meta 공식 발표: Andromeda는 "오가닉 피드에 자연스러운 광고"를 더 많이, 더 싸게 노출시킨다. 피드 네이티브일수록 CPM 하락 + 노출 증가.

  • 첫 3초가 전부 — 비주얼 첫 프레임이 오가닉 피드와 얼마나 비슷한가
  • 텍스트 오버레이 자연스러움 — "광고처럼 생겼냐 vs 일반 게시물처럼 생겼냐"
  • 색감/톤 일치 — 타겟 사용자의 피드 색감과 얼마나 매칭되는가
  • 로고/CTA 노출 시점 — 너무 초반에 광고임을 드러내면 Andromeda가 감점
Motion에 없는 기능 — bscamp만의 차별점

🟢 Andromeda Score 설계

L1 분석에 andromeda_score 필드 추가. 0~100점 구조.

평가 항목 가중치 평가 방법
오가닉 피드 유사도 35점 타겟 오가닉 피드 레퍼런스 이미지 vs 소재 첫 프레임 임베딩 코사인 유사도
텍스트 오버레이 자연스러움 25점 텍스트 위치/크기/폰트가 일반 게시물 수준인지 (광고 폰트 = 감점)
색감/톤 일치도 20점 소재 색상 팔레트 vs 타겟 피드 평균 색상 유사도
첫 프레임 Visual Native-ness 20점 Gemini로 "첫 프레임이 일반 사용자 게시물처럼 보이는가" 평가 
-- andromeda_score 계산 (L1 분석 결과에 추가)
UPDATE creative_element_analysis
SET andromeda_score = (
  feed_similarity_score * 0.35 +
  text_naturalness_score * 0.25 +
  color_match_score * 0.20 +
  visual_native_score * 0.20
) * 100
WHERE creative_id = $1;

활용 시나리오:

  • "Andromeda 점수 80점 이상 소재의 평균 CPM이 40% 낮다" → 수강생한테 "피드 네이티브 소재 만드는 법" 가이드
  • 소재 제출 시 Andromeda 점수 실시간 피드백 → "이 텍스트가 너무 광고스럽습니다. 크기를 줄여보세요"
🔥 이 기능은 Motion도 구현 안 한다. "Andromeda 최적화 점수" — bscamp만의 차별점. 다른 어떤 광고 분석 서비스도 없는 것. 수강생한테 "왜 내 광고가 비싸게 노출되는지" 데이터로 설명하는 유일한 도구.

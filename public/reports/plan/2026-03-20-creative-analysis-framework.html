Meta 내부 소재 분석 시스템
Andromeda · GEM · PE-AV — Meta가 소재를 어떻게 분석하는지, 우리가 어떻게 따라가는지.
💡 핵심 개념: Entity ID (시맨틱 지문)
Meta는 소재가 업로드되면 Entity ID를 자동 생성한다. 비슷한 Entity ID를 가진 소재끼리 클러스터링하여 같은 광고로 취급한다. 중복 소재가 많을수록 경매에서 불리해진다.
Entity ID 생성 요소
🖼
Visual Similarity
픽셀 구성, 색상 팔레트, 조명 패턴, 레이아웃 구조, 배경 요소, 인물/제품 위치·포즈·크기
→ 같은 스타일 제품 사진 전부 = 같은 Entity ID
📐
Structural Similarity
텍스트 배치 좌표, CTA 버튼 위치, 오버레이 구조, 프레임 구성
→ 사진 달라도 레이아웃 동일 = 같은 Entity ID
💬
Thematic Similarity
메시지 컨셉, 가치 제안(VP), 감정 톤, 타겟 고충(pain point)
→ 비주얼 달라도 "같은 말" = 같은 Entity ID
🎬
Motion Pattern VIDEO
동영상 내 움직임, 씬 전환 속도, 카메라 워크
→ 모션 패턴 비슷하면 유사 판정
🎵
Audio Signature VIDEO
나레이션 톤, BGM 장르, 사운드 디자인
→ 향후 더 정교해질 예정

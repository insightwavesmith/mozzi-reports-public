1. 개요

롱폼 유튜브 영상 + SRT 자막 파일을 입력하면 AI가 하이라이트 구간을 선정하고 자동으로 쇼츠 5개를 생성하는 로컬 웹앱. 별도 서버 없이 Mac/PC에서 바로 실행 가능. 카드뉴스 모드를 통해 같은 입력으로 인스타그램 카드뉴스 이미지도 동시 생성.

🎬 레퍼런스: 커리어해커 알렉스

youtube.com/watch?v=EUi09O_b7so — Claude Code로 1시간 만에 라이브 개발. 같은 방식으로 우리 버전 구현 가능.

💡 핵심 가치: 영상 1개 업로드 → 클릭 1번 → 쇼츠 5개 + 카드뉴스 7장 자동 생성. 편집 시간 90% 절감.

4. AI 프롬프트 설계 (숏츠)

하이라이트 선정 프롬프트

System Prompt → Claude Sonnet
이 자막에서 숏폼 영상으로 만들기 좋은 구간 5개를 골라줘.

선정 기준:
- 흥미로운 도입부 (호기심 유발)
- 핵심 인사이트 (한 문장으로 배울 수 있는 것)
- 임팩트 있는 한 마디 (공유하고 싶은 말)
- 독립적으로 이해 가능한 구간 (맥락 없이도 통하는)
- 30초-60초 길이

각 구간의 시작 타임스탬프, 끝 타임스탬프, 제목, 선정 이유를 JSON으로 반환.

응답 포맷

// Claude API 응답 스키마
{
  "clips": [
    {
      "title": "프롬프트 엔지니어링은 끝났다",
      "start": "00:02:15",
      "end": "00:03:05",
      "reason": "강한 도입부 + 핵심 메시지"
    },
    // ... 총 5개
  ]
}

✅ 프롬프트 설계 포인트

  • 선정 기준 5가지를 명시 → 다양한 유형의 클립 확보 (도입부, 인사이트, 명언 등)
  • 30~60초 길이 제약 → 숏츠 플랫폼 최적 길이
  • 독립 이해 가능 조건 → 맥락 없이도 통하는 클립만 선정
  • JSON 반환 → ffmpeg 파이프라인 직접 연결 가능

5. ffmpeg 처리 파이프라인

1
구간 추출
Claude가 선정한 시작/끝 타임스탬프로 원본 영상에서 클립 추출. 재인코딩 없이 복사 → 빠른 처리.
2
세로 크롭
16:9 가로 영상 → 9:16 세로 변환. 중앙 크롭 방식 (ih*9/16:ih). 얼굴/자막이 중앙에 있으면 OK.
3
자막 하드코딩 (선택)
원본 SRT에서 해당 구간만 추출해서 영상에 소각. 쇼츠 자막 효과로 시청 유지율 향상.
4
최종 출력
YouTube Shorts / TikTok / Instagram Reels 업로드 가능한 MP4 포맷. H.264, AAC 인코딩.

ffmpeg 명령어

# 1단계: 구간 추출 (빠른 복사)
ffmpeg -ss {start} -to {end} -i input.mp4 -c copy clip_{n}.mp4

# 2단계: 세로 크롭 (9:16)
ffmpeg -i clip_{n}.mp4 -vf "crop=ih*9/16:ih" -c:a copy shorts_{n}.mp4

# 3단계: (선택) 자막 하드코딩
ffmpeg -i shorts_{n}.mp4 -vf "subtitles=clip_{n}.srt" final_{n}.mp4

⚠️ 주의사항

  • 1단계에서 -c copy는 재인코딩 없이 복사 → 빠르지만 정확한 타임스탬프 컷은 2패스 필요
  • 세로 크롭 시 피사체가 중앙에 없으면 얼굴이 잘릴 수 있음 → Phase 4에서 스마트 크롭(얼굴 감지) 옵션 추가 가능
  • 자막 하드코딩 시 폰트 설정 필요 (한글 깨짐 방지)

📱 UI 컴포넌트 목록

  • 모드 선택 토글 — 숏츠 모드 / 카드뉴스 모드 / 둘 다 (3선택)
  • 드래그앤드롭 영역 — mp4 + srt 파일 동시 수용. 텍스트 입력 / URL 입력 탭도 제공.
  • Claude API 키 입력 필드 — localStorage 저장 (매번 입력 불필요)
  • 분석 시작 버튼 — 파일/텍스트 + API 키 있을 때 활성화
  • 진행률 표시 — 파싱 → AI 분석 → 클리핑/렌더링 → 완료 (단계별 표시)
  • [숏츠] 결과 카드 5개 — 썸네일 + 제목 + 길이 + 선정 이유 + 다운로드
  • [카드뉴스] 캐러셀 미리보기 — 7장 슬라이드 + 전체 ZIP 다운로드
  • 전체 다운로드 버튼 — 숏츠 ZIP or 카드뉴스 ZIP

UI 흐름

  1. 모드 선택 → 입력 영역 변경 (영상+SRT / 텍스트 붙여넣기 / URL 입력)
  2. 파일/텍스트 입력 → 유효성 검증 표시
  3. API 키 입력 (처음 한 번 → 이후 자동 불러오기)
  4. "분석 시작" 클릭 → 로딩 스피너 + 단계별 진행 메시지
  5. 완료 → 결과 카드(숏츠) 또는 캐러셀(카드뉴스) 애니메이션 등장
  6. 카드별 미리보기 + 개별/전체 다운로드

7. 카드뉴스 자동생성 모드 ✨

기존 숏츠 모드와 동일한 입력(롱폼 영상 + SRT 또는 텍스트만)으로 인스타그램 카드뉴스 이미지 7장을 자동 생성. 영상 없이 텍스트나 URL만으로도 동작.

🃏 카드뉴스 가치: 강의 1개 → 카드뉴스 7장 자동 완성. 인스타 피드 1주일치 콘텐츠 10분 만에 해결.

카드뉴스 AI 프롬프트

Card News Prompt → Claude Sonnet
이 콘텐츠에서 인스타그램 카드뉴스로 만들기 좋은 핵심 포인트 7개를 뽑아줘.

각 카드 구성:
- 카드 1: 표지 (후킹 제목 1줄)
- 카드 2-6: 본문 (각각 제목 1줄 + 설명 2-3줄)
- 카드 7: 마무리 (CTA 또는 요약)

JSON으로 반환: 
// 카드뉴스 Claude API 응답 스키마
{
  "cards": [
    {"type": "cover", "title": "체험단 소재, 승인해야 할까?"},
    {
      "type": "content",
      "title": "기준은 딱 3가지",
      "body": "1. 제품이 메인으로 보이는지\n2. 실사용 느낌이 나는지\n3. 광고 소재로 가공 가능한지"
    },
    {
      "type": "cta",
      "title": "일단 돌려보세요",
      "body": "70점짜리 10개가 100점짜리 1개보다 낫습니다"
    }
  ]
}

추가 입력 지원

📌 3가지 입력 방식

  • 영상 + SRT (기존) — 롱폼 영상 + 자막 파일로 숏츠와 카드뉴스 동시 생성

9. 구현 로드맵

총 예상 개발 시간: 9시간 (Phase 5~6 포함, Phase 4 제외)

  • Phase 1
    2시간
    FastAPI 서버 + SRT 파서 + Claude API 연동
    서버 뼈대 구축. SRT 파일 파싱 → 텍스트+타임스탬프 추출. Claude API 호출 → 하이라이트 5개 JSON 반환 → 로컬 저장. 기본 API 엔드포인트: POST /analyze
  • Phase 2
    2시간
    ffmpeg 클리핑 + 세로 변환
    Phase 1 결과(JSON) → ffmpeg 명령어 생성 → 자동 실행. 구간 추출 → 9:16 크롭 → MP4 출력. API 엔드포인트: POST /process, SSE로 진행률 스트리밍.
  • Phase 3
    1시간
    웹 UI (드래그앤드롭 + 결과 표시)
    단일 HTML 파일. 드래그앤드롭 → API 호출 → 진행률 표시. 완료 시 결과 카드 5개 + video 미리보기 + 다운로드 버튼. 모드 선택 토글 (숏츠 / 카드뉴스 / 둘 다) UI 포함. FastAPI에서 static 파일로 서빙.
  • Phase 4
    선택
    자막 하드코딩 + YouTube 업로드 API
    자막 소각으로 시청 유지율 향상. YouTube Data API v3로 자동 업로드. 스마트 크롭(얼굴 감지) 옵션. bscamp 플랫폼 통합.
  • Phase 5
    2시간
    카드뉴스 모드 — Claude 프롬프트 + HTML Canvas 렌더링 + PNG 내보내기
    카드뉴스용 Claude 프롬프트 구현 → JSON 카드 데이터 반환. HTML Canvas API로 1080×1350px 카드 렌더링 (표지/본문/CTA 타입별 레이아웃). Pretendard 폰트 + bscamp 브랜드 컬러 적용. PNG 개별 다운로드 + JSZip으로 전체 ZIP 내보내기. 캐러셀 미리보기 UI (좌우 화살표 슬라이드).
  • Phase 6
    1시간
    텍스트 / URL 입력 지원
    텍스트 붙여넣기 입력 탭 추가 (영상 없이 카드뉴스 생성 가능). URL 입력 → 서버에서 웹페이지 본문 추출 (BeautifulSoup) → 카드뉴스 변환. 블로그 글, 강의 스크립트, Q&A 텍스트 직접 입력 지원.

10. 필요 리소스

항목 상태 비고
Python 3.9+ ✅ 준비됨 FastAPI, srt, anthropic, beautifulsoup4 패키지 pip 설치
ffmpeg ✅ 준비됨 homebrew 설치 완료
Claude API Key ✅ 준비됨 ANTHROPIC_API_KEY 환경변수 (숏츠 + 카드뉴스 공용)
Node.js (웹 UI) ✅ 준비됨 빌드 도구 필요 시 (현재는 순수 HTML + Canvas API로 충분)

Pretendard 폰트 ✅ CDN 사용 Canvas FontFace API로 로드. 한글 카드 렌더링 필수.
YouTube Data API ⬜ Phase 4 자동 업로드 기능 구현 시 필요

pip install fastapi uvicorn srt anthropic python-multipart beautifulsoup4 requests
uvicorn main:app --reload --port 8000

브라우저에서 http://localhost:8000 접속 → 드래그앤드롭 UI + 모드 선택 토글 확인

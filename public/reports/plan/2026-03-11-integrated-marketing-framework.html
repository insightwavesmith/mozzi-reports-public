1 글로벌 대가들의 핵심 프레임워크

① Alex Hormozi — Value Equation $100M Offers, 2021

Value = (Dream Outcome × Perceived Likelihood) / (Time Delay × Effort)
가치 = (꿈의 결과 × 실현 가능성) / (시간 지연 × 투입 노력)

Hormozi의 핵심: 가격이 아니라 가치로 승부. 경쟁사가 가격을 낮출 때, 우리는 가치 방정식의 분자를 올리고 분모를 낮춘다. "거부할 수 없는 오퍼"란 가치가 가격을 압도하는 상태다.

변수의미화장품 적용 예시방향
Dream Outcome고객이 원하는 최종 결과"3일 만에 톤업" / "촉촉한 아기피부"↑ 극대화
Perceived Likelihood실현될 것 같은 확신"500만명이 검증" / "피부과 의사 추천"↑ 극대화
Time Delay결과까지 걸리는 시간"세안 후 즉시 흡수" / "첫날부터 차이"↓ 최소화
Effort고객의 투입 노력"세안 후 1단계로 끝" / "30초 루틴"↓ 최소화
엠타트업 연결: Hormozi Value Equation = 엠타트업 등가교환 + 구매편익 4분류. "어떤 가치를" 제공할지는 구매편익 4분류로, "가치가 가격을 압도하는지"는 등가교환으로 점검한다.

② Eugene Schwartz — 5 Stages of Awareness Breakthrough Advertising, 1966

1966년에 쓰였지만 지금도 광고 카피라이팅의 성경. 핵심: 같은 제품도 고객이 어느 인지 단계에 있느냐에 따라 카피 전략이 완전히 달라야 한다.

레벨5 · Unaware 문제 자체를 모름 카피: 공감할 수 있는 이야기로 문제를 처음 제시 예: "요즘 거울 볼 때마다 피부가 칙칙해 보이지 않나요?" ───────────────────────────────────────────────────── 레벨4 · Problem Aware 문제는 알지만 해결책 모름 카피: 문제를 명확히 정의, 공감 + 긴박감 예: "피부 톤이 고르지 않은 건 색소침착+산화 때문입니다" ───────────────────────────────────────────────────── 레벨3 · Solution Aware 해결책 카테고리는 알지만 제품 모름 카피: 우리 카테고리가 최선임을 증명 예: "비타민C 세럼이 톤업에 가장 빠른 이유 5가지" ───────────────────────────────────────────────────── 레벨2 · Product Aware 제품 알지만 아직 구매 결정 안 함 카피: 경쟁 제품 대비 명확한 차별화 예: "왜 OOO 세럼이 다른 비타C보다 3배 빠를까" ───────────────────────────────────────────────────── 레벨1 · Most Aware 살 준비 됨, 트리거만 필요 카피: 오퍼 + 긴박감 + CTA 예: "오늘만 30% 할인 + 무료배송 + 사은품. 마감 3시간"
🔑 핵심 발견 — 엠타트업 키워드 5단계와 1:1 매핑
Schwartz 단계엠타트업 키워드 레벨검색 키워드 예시 (화장품)
Unaware (5)레벨5 · 순수정보"피부 관리 방법", "피부 좋아지는 법"
Problem Aware (4)레벨4 · 문제인식"피부 칙칙한 이유", "잡티 생기는 원인"
Solution Aware (3)레벨3 · 대안탐색"미백 세럼 효과", "톤업 방법 추천"
Product Aware (2)레벨2 · 비교검토"OOO 세럼 후기", "비타C 세럼 비교"
Most Aware (1)레벨1 · 구매직전"OOO 세럼 구매", "OOO 세럼 최저가"

③ Russell Brunson — Value Ladder + Sales Funnel DotCom Secrets, 2015

Continuity $$$$ 정기구독 (월정액/자동결제) ← LTV 극대화 ▲ Backend $$$ 프리미엄 세트 / 상위 서비스 ▲ Middle $$ 번들 세트 / 패키지 ▲ Frontend $ 단품 첫 구매 (저위험 진입) ▲ Bait FREE 샘플 / 무료 체험 / 콘텐츠 ← 첫 접촉

고객을 한 번에 고가로 끌어올리려 하지 마라. 작은 신뢰를 쌓고 단계적으로 올린다. 첫 거래에서 등가교환(가치>가격)을 체감시키면 다음 단계 설득이 쉬워진다.

단계화장품 적용목적예상 전환율
Bait샘플 키트 (배송비만 부담)첫 접촉, DB 확보15~30%
Frontend단품 30ml (첫 구매 할인)첫 구매 경험5~15%
Middle2종 세트 / 루틴 패키지객단가 2× 상승20~35%
Backend프리미엄 라인 / VIP 세트고가치 고객 전환5~10%
Continuity정기구독 (월 1회 자동 배송)LTV 극대화10~20%

④ Ryan Deiss — Customer Value Journey DigitalMarketer

1. Aware 브랜드/제품 존재 인식 (광고/SNS/추천) 2. Engage 콘텐츠/광고와 상호작용 (좋아요, 댓글, 공유) 3. Subscribe허락 마케팅 채널 가입 (이메일, 카카오, 문자) 4. Convert 첫 구매 — Tripwire (저가, 저위험, 와우 경험 설계) 5. Excite "이 브랜드 진짜다!" 와우 경험 → 신뢰 폭발 6. Ascend 업셀/크로스셀 → 객단가 + LTV 상승 7. Advocate 만족 → 자발적 후기/리뷰/추천 8. Promote 적극적 홍보 → 인플루언서/앰배서더화

구매 이후가 더 중요하다. 5번 Excite(와우 경험)가 실패하면 7, 8번은 없다. 자사몰사관학교의 "첫구매+재구매 순환루프"가 CVJ 4→8단계 구현이다. LTV = 객단가 × 구매빈도 × 고객수명.

⑤ Seth Godin — Purple Cow + Permission Marketing Purple Cow, 2003

Purple Cow (보랏빛 소):
평범한 소 수천 마리 사이에서 보랏빛 소 한 마리가 지나가면 모두가 멈춰서 본다.
리마커블(Remarkable)하지 않으면 아무도 주목하지 않는다. 평범한 제품은 광고해봤자 무의미.

Permission Marketing (허락 마케팅):
고객의 허락(Permission)을 얻은 다음 소통한다. 이메일 구독, 카카오 채널 팔로워, 앱 푸시 수신 동의 = Permission.
반대: Interruption Marketing (강제로 끼어드는 광고) — 효율이 계속 하락 중.

AI 시대에 Purple Cow는 더 중요해진다. 매일 수백만 개의 콘텐츠가 쏟아지는 환경에서 평범한 제품/콘텐츠는 알고리즘에 묻힌다. 엠타트업의 "가치더하기"가 바로 Purple Cow 만들기다.

⑥ Sabri Suby — 8 Phase Selling System Sell Like Crazy, 2019

시장 분류 (Suby의 핵심 인사이트):
비율단계특징접근 전략
3%구매 준비지금 당장 살 준비 됨직접 오퍼 → 즉시 전환
17%정보 수집구매 고려, 비교 중교육 + 차별화 + 신뢰
20%문제 인식문제는 알지만 해결책 탐색 전문제 정의 + HVCO
60%무인식문제 자체를 모름어웨어니스 콘텐츠

Godfather Offer: 거부할 수 없는 제안 설계 — Hormozi의 Value Equation과 동일한 철학. HVCO(High Value Content Offer): 무료 가치 제공 → 리드 확보 → 너처링 → 전환. 엠타트업의 O-M-C(오프닝-메리트-클로징)와 구조적으로 동일하다.

Phase 1 HVCO 설계 (무료 가치 제공) Phase 2 리드 페이지 (랜딩페이지) Phase 3 트래픽 → 리드 전환 Phase 4 이메일/SMS 너처링 시퀀스 Phase 5 판매 페이지 + Godfather Offer Phase 6 전화/채팅 클로징 (고관여) Phase 7 재구매 + 업셀 이메일 Phase 8 리퍼럴 + 리뷰 수집

⑦ AARRR — Pirate Metrics Dave McClure, 500 Startups

A → A → R → R → R
Acquisition(획득) → Activation(활성화) → Retention(유지) → Revenue(매출) → Referral(추천)
단계의미이커머스 지표자사몰 연결
Acquisition어떻게 사람들이 오는가신규 방문자, CPC, CTR트래픽 + 광고 CTR
Activation첫 좋은 경험을 하는가첫 방문 체류시간, 장바구니 담기체류 × 결제시작
Retention다시 돌아오는가재방문율, 재구매율재구매 순환루프
Revenue돈을 버는가구매전환율, ROAS, LTV곱셈의 법칙 최종값
Referral추천하는가추천인 수, 바이럴 계수Advocate → Promote
자사몰사관학교 "곱셈의 법칙"과 연결: 각 단계를 10%씩 개선하면 전체 성과는 46% 상승. AARRR 각 단계 = 곱셈의 법칙 각 단계와 1:1 대응.

2 엠타트업 핵심 프레임워크

마케팅 = 모으고(Resource/Channel) + 설득하기(System). 이 두 단어 안에 마케팅의 모든 것이 들어있다.

대전제: RCS 프레임워크

구성의미핵심 질문
R — Resource우리가 가진 것"우리 제품/브랜드의 핵심가치는 무엇인가?"
C — Channel고객을 모으는 채널"어디서, 어떻게 고객을 만나는가?"
S — System설득 → 구매 → 재구매 시스템"첫 구매 후 어떻게 관계를 유지하는가?"

R: 리소스 평가 5기준 (VRIN+M)

기준질문평가 방법
시장성 (Value)시장이 원하는 리소스인가?검색량, 시장 규모, 트렌드
희귀성 (Rarity)경쟁사가 없거나 적은가?경쟁사 수, 차별화 가능성
모방성 (Imitability)따라 하기 어려운가?특허, 노하우, 원료, 공정
접근성 (Non-substitutability)대체재가 없는가?유사 제품 비교
가치성 (Margin)수익이 나는가?원가율, 예상 마진

핵심가치(CV) 3종류 + 핵심가치 이동

CV1: 기능가치 제품 자체의 기능적 성능 예: "SPF50+ 자외선 차단", "24시간 보습 지속" CV2: 경험가치 사용하는 과정의 감각적 경험 예: "발림감이 가볍다", "향이 기분 좋다" CV3: 상징가치 제품이 상징하는 이미지/신분 예: "프리미엄 스킨케어 유저", "환경 의식있는 소비" 핵심가치 이동: 시장이 성숙할수록 CV1 → CV2 → CV3으로 경쟁 축이 이동 스마트폰: 초기(기능) → 카메라(경험) → 브랜드(상징) 화장품: 성분(기능) → 발림감(경험) → 라이프스타일(상징)

구매편익 4분류 + 승패 관계

편익의미예시경쟁 관계
기능편익실용적 효과/성능"미백, 수분, 자외선 차단"기능↔경험 대체
경험편익사용 과정의 즐거움/감각"발림감, 향, 패키지 개봉 경험"
상징편익사회적 이미지/소속"OOO 쓰는 사람 = 세련된 사람"상징↔경제 대체
경제편익가격/가성비/절약"1+1, 대용량 리필, 구독 할인"
승패 관계: 같은 편익 카테고리 내에서 경쟁해야 이긴다. 기능이 약한 제품이 기능으로 싸우면 진다. 차별화 불가 시 상징편익이나 경험편익으로 경쟁 축을 바꿔라.

마케팅축 — 욕구 / 대안 / 접근

욕구축: 고객이 원하는 것 (3대 욕구) → 생리적 욕구 (건강, 생존, 안전) → 사회적 욕구 (소속, 인정, 관계) → 자아실현 욕구 (성장, 성취, 표현) 대안축: 고객의 선택지 (우리 vs 경쟁사 vs 대체재) → 직접 경쟁자: 동일 카테고리 → 간접 경쟁자: 다른 방식으로 같은 욕구 해결 → 대체재: 아예 다른 방법 접근축: 고객에게 닿는 방법 → 어디서? (채널) → 언제? (타이밍) → 어떻게? (메시지/형식)
욕구깡패: 3대 욕구 중 가장 강한 것이 구매를 지배한다. 화장품은 표면적으로는 기능(생리) 욕구지만, 실제 구매는 사회적 욕구(타인에게 예뻐 보이고 싶다)가 지배하는 경우가 많다. 마케팅 메시지는 진짜 욕구를 건드려야 한다.

관여도 스펙트럼 + 관여도 이동

고관여 ←────────────────────→ 저관여 집/차/보험/B2B 솔루션 일상 소모품/간식/소액 앱 (깊이 생각하고 구매) (거의 생각 없이 구매) 관여도가 높을수록: 정보 제공 → 비교 → 신뢰 구축이 중요 관여도가 낮을수록: 노출 빈도 → 감정 → 즉각 구매가 중요 관여도 이동 전략: → 저관여를 고관여처럼: 이야기/철학으로 관여도 높이기 → 고관여를 저관여처럼: 구매 장벽 낮추기 (분할결제, 체험)

포지셔닝 + 프레임 분리

포지셔닝: 고객의 마음속 특정 위치를 선점하는 것.
"OOO하면 OOO" — 1등이 되는 영역을 스스로 정의한다.
예: "데일리 톤업 전문" / "민감성 피부 첫 번째 선택" / "성분 투명 브랜드"

프레임 분리: 기존 카테고리에서 새 프레임을 만들어 혼자 1등이 되는 전략.
예: "일반 세럼" 프레임 → "아침 전용 세럼" 프레임 분리 → 새 카테고리 1등

가치더하기 (Value Addition)

제품에 직접적 가치 외에 부가 가치를 더해 경쟁에서 이탈하는 전략. Purple Cow 만들기의 실전 도구.

가치더하기 유형예시
정보/교육 더하기성분 설명 카드, 사용법 영상, 피부 타입별 가이드
경험 더하기언박싱 경험 설계, 향기, 패키지 디자인
관계 더하기개인화 메시지, 구독자 커뮤니티, 브랜드 스토리
보증 더하기30일 환불 보장, 평생 교환 보장
편의 더하기정기구독, 자동 리필, 새벽배송

접촉지점 + 키워드 5단계

고객이 제품을 알게 되는 모든 접촉지점 (Touchpoint) 매핑: 인지 전 SNS 피드, 유튜브 광고, 추천 알고리즘 인지 후 검색 (네이버/구글), 리뷰 플랫폼 비교 중 상세페이지, 경쟁사 비교 블로그 구매 직전 장바구니, 결제 페이지, 쿠폰 구매 후 배송 알림, 패키지, 고객센터, 재구매 CRM 키워드 5단계 = Schwartz 매핑 (섹션1 참조)

선택제어 3종

기법원리예시
이지선다2가지 선택지만 제시 → 둘 다 우리한테 유리"지금 구매하시겠어요, 아니면 샘플 받아보시겠어요?"
더블바인드어떤 선택을 해도 긍정적 결과 → 저항 해제"오늘 주문하면 내일 도착, 내일 주문해도 빠른배송"
소거법대안을 하나씩 제거 → 우리만 남음"타 브랜드는 이 성분이 없고... 결국 OOO만이"

O-M-C 설득 구조 + 클로징 4요소

O — Opening 관심 끌기 + 문제 제시 3초 Hook → 공감 → 긴박감 형성 M — Merit 가치 증명 혜택 설명 → 증거 제시 → 반론 해소 C — Closing 행동 유도 4요소: ① 명확한 CTA ② 긴박감(마감/수량) ③ 리스크 제거(환불 보장) ④ 보너스(사은품)

등가교환 + 매몰비용

등가교환: 고객이 지불하는 가격과 받는 가치가 같거나 가치가 더 크다고 느낄 때 구매한다.
가치 > 가격 체감 = 구매 발생. 가치 < 가격 체감 = 이탈.
마케팅의 역할은 가치를 제대로 보이게 만드는 것.

매몰비용: 이미 투자한 것이 있으면 더 투자하는 경향.
샘플 → 단품 → 세트 → 구독: 각 단계에서 이전 경험이 매몰비용으로 작용 → 다음 단계 전환율 상승.

물길(트래픽) 프레임워크

상류 (Source): 트래픽이 발생하는 곳 - 유료: Meta 광고, 구글 광고, 네이버 광고 - 자연: SEO, SNS, 유튜브, 카페 - 소개: 리퍼럴, 인플루언서 중류 (Flow): 트래픽이 흐르는 경로 - 랜딩페이지 → 상세페이지 → 장바구니 → 결제 하류 (Pool): 트래픽이 모이는 곳 - 자사몰, 스마트스토어, 쿠팡 - CRM DB (이메일, 카카오, 문자) 물길 전략: 상류를 다양화, 중류 누수(이탈) 최소화, 하류에 자산 축적

배제와 양보의 설득법

배제 설득: "이 제품은 아무에게나 권하지 않습니다. OOO한 분들만." → 희소성 + 자격 부여.

양보 설득: "가격이 비싼 건 맞아요. 그런데..." → 단점을 먼저 인정하면 신뢰가 올라가고 그 다음 메시지의 설득력이 강해진다.

제로 마케팅

광고비 없이(또는 최소한으로) 마케팅하는 전략. SNS 자연 유입, 블로그 SEO, 커뮤니티 마케팅, 바이럴. 자원이 적은 초기 브랜드에 필수. 나중에 유료 광고를 쓸 때도 제로 마케팅 채널이 신뢰도를 높인다.

브랜드 4유형 + 네이밍

유형특징예시전략

회사 브랜드회사 이름이 브랜드삼성, LG, 애플신뢰 + 라인업 확장

네이밍 원칙: ① 기억하기 쉬울 것 ② 발음 쉬울 것 ③ 의미 있을 것 ④ 검색 가능할 것 ⑤ 상표 등록 가능할 것. AI(Claude)로 수백 개 후보 생성 후 5원칙으로 필터링.

시스템 7대 영역

1. 유입 시스템 새 고객이 계속 들어오는 구조 (광고/SEO/바이럴) 2. 전환 시스템 방문자 → 구매자 전환 (상세페이지/UX/오퍼) 3. 재구매 시스템 한 번 산 사람이 다시 사는 구조 (CRM/구독) 4. 객단가 시스템 한 번 살 때 더 많이 사는 구조 (업셀/번들) 5. 추천 시스템 고객이 고객을 데려오는 구조 (리퍼럴/리뷰) 6. 운영 시스템 CS, 물류, 재고, 품질 관리 7. 데이터 시스템 위 6개를 측정하고 개선하는 구조

통찰력 = 5분 안에 마케팅 그림 그리기

어떤 브랜드/제품을 봐도 5분 안에 RCS 그림을 그릴 수 있는 능력. Resource 평가 → 핵심가치 정의 → 경쟁 포지셔닝 → 채널 전략 → 시스템 구조. 이 능력이 커리큘럼 최종 목표.

3 자사몰사관학교 핵심 프레임워크

Meta 광고는 기계다. 기계가 잘 작동하려면 올바른 데이터(소재+타겟+구조)를 제대로 넣어야 한다. 자사몰사관학교는 그 방법을 가르친다.

총가치 공식

총가치 = 광고비 × 추산행동률 + 광고품질
Meta가 광고주에게 노출 기회를 배분하는 경매 공식
광고비가 많아도 추산행동률(예상 CTR × CVR)이 낮으면 경매에서 진다.
광고품질(소재의 관련성 + 참여도)이 높으면 적은 광고비로 많은 노출을 얻는다.
결론: 좋은 소재 = 광고비 절약 + 더 많은 노출 = 선순환

주인공 정의: 작가시점 ∩ 고객시점

작가시점 (브랜드가 생각하는 주인공) → 우리 제품의 이상적 고객은 누구인가? → 어떤 삶을 살고, 어떤 고민이 있는가? 고객시점 (실제 고객이 생각하는 자기 자신) → 고객은 스스로를 어떻게 정의하는가? → 어떤 욕구가 있고 어떤 언어를 쓰는가? 주인공 = 작가시점 ∩ 고객시점 (교집합) → 두 시점이 겹치는 지점 = 진짜 주인공 → 이 지점에서 만들어진 소재가 공명한다

3장면 구조

장면역할목적예시
문제장면 (Hook)주인공의 현재 고통공감 → 멈춤"요즘 거울 보기 싫지 않으세요?"
전환장면 (아하!)전환점 — 제품 등장아하! 순간"이 성분 하나가 달랐어요"
이상장면 (기대감)제품 사용 후 이상적 미래구매 욕구 자극"3일 후 거울 보는 나의 표정"

싱귤래리티 효과 + 스노우볼 효과

싱귤래리티 효과 (Singularity Effect):
소재 하나가 완벽하게 타겟에 공명할 때 발생하는 폭발적 성과. 같은 예산으로 100배 차이가 난다.
→ 좋은 소재 1개 > 평범한 소재 100개

스노우볼 효과:
좋은 소재 → Meta 머신러닝이 더 많이 노출 → 더 많은 데이터 수집 → 더 정확한 타겟팅 → 더 좋은 성과.
선순환이 시작되면 가속도가 붙는다.

Meta 경매 4단계

1. 필터링 광고 정책 위반 여부 검사 (자동화) 2. 리트리벌 광고주 DB에서 후보 광고 수천 개 추출 3. 시퀀스러닝 후보 광고별 예상 성과(CTR×CVR) 랭킹 4. 라티스 최종 경매 — 총가치 공식으로 노출 배분 → 시퀀스러닝이 핵심: ML이 "이 소재가 이 사람에게 효과있을까?" 예측 → 데이터(행동지문)가 많을수록 예측 정확도↑

행동지문 = 성향 + 취향 + 행동 = 광고 임베딩

Meta는 모든 사용자의 행동 패턴을 벡터로 임베딩한다. 좋아요, 댓글, 구매, 검색, 체류시간 — 모든 행동이 지문이 된다. 광고 소재도 마찬가지로 임베딩된다. 소재와 사용자의 임베딩 유사도가 높을수록 노출 확률이 올라간다.

곱셈의 법칙

ROAS = CTR × 체류율 × 결제시작률 × 구매전환율
각 단계가 독립 변수 — 하나가 0이면 전체가 0
단계지표낮으면 문제벤치마크
CTR광고 클릭률소재/Hook이 약함1% 이상
체류율랜딩 후 3초 이상 머묾상세페이지 도입이 약함60% 이상
결제시작률장바구니 → 결제 시작가격/신뢰 장벽30% 이상
구매전환율결제 완료복잡한 결제 UX, 불안감70% 이상

배수효과 진단: 벤치마크 vs 현재값

각 단계 현재값을 벤치마크와 비교 → 가장 낮은 단계가 병목 → 거기에 자원 집중. CTR이 0.3%면 소재가 문제. 결제시작률이 낮으면 가격/신뢰 문제. 진단 없이 광고비만 늘리면 낭비.

캠페인 구조 4원칙 + ASC+

원칙1: 캠페인 = 목적. 하나의 캠페인은 하나의 목적만. 원칙2: 광고세트 = 예산 + 타겟. A/B 테스트 단위. 원칙3: 광고 = 소재. 소재 다양성 확보 (3~5개+). 원칙4: 데이터 축적 우선. 섣부른 수정 금지 (최소 3~7일). ASC+ (Advantage+ Shopping Campaign): → Meta ML에 최대한 자율성 부여 → 타겟 설정 최소화 → ML이 최적 타겟 탐색 → 소재 다양성이 핵심 — 최소 10개 이상 권장

6 AI 활용 가이드 (Phase별)

AI는 마케터를 대체하지 않는다. AI는 마케팅 사고력이 있는 사람을 10배 빠르게 만든다. 프레임워크를 먼저 이해하고, 그 다음 AI로 가속한다.

Phase 1 (1~4주): Claude 기초 — 리서치 & 분석

활용 영역프롬프트 예시산출물
시장 리서치"[카테고리] 시장의 주요 플레이어 10개, 각각 포지셔닝과 주요 메시지 분석해줘"경쟁사 분석표
고객 분석"[제품] 구매하는 30대 여성의 페르소나를 상세하게 작성해줘. 일상/고민/미디어 소비 포함"페르소나 3개
키워드 발굴"[제품]의 Schwartz 5단계별 검색 키워드를 각 10개씩 추출해줘"키워드 맵 50개
리뷰 분석"아래 고객 리뷰 50개를 분석해서 욕구/불만/기대를 분류해줘: [리뷰 붙여넣기]"욕구 분류표

Phase 2 (5~8주): Midjourney + Canva — 소재 제작

도구활용 방법팁

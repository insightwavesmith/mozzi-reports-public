v1과 v2의 핵심 차이 — 계획에서 실행으로

v1(2/17)은 "7채널 완전 자동화"라는 이상적 마스터 플랜이었다. 유튜브 쇼츠 자동 생성, 인스타 캐러셀 변환, 카카오 친구톡까지 — 범위가 넓었다.

v2는 다르다. 지금 가진 인프라로, 지금 효과가 나는 채널에 집중한다.
네이버 블로그/카페(검색 유입) + 구글 SEO(글로벌 권위) + GEO(AI 추천) + 뉴스레터(기존 리드 육성).
이 4개 채널만 제대로 돌리면, 유튜브/인스타는 콘텐츠가 쌓인 뒤 자연스럽게 확장된다.

왜 4채널 집중인가

핵심 논리: "메타 광고 교육"을 찾는 사람은 검색한다. 인스타 릴스를 보다가 "오 이거 배워야지"가 아니라, "메타 광고 ROAS 올리는 법"을 네이버에서, 구글에서, ChatGPT에서 검색한다. 검색 채널을 먼저 장악하는 것이 퍼널 TOFU의 본질이다.

소셜(유튜브 쇼츠/인스타 릴스)은 "발견"이지 "탐색"이 아니다. 발견형 채널은 콘텐츠 볼륨이 나와야 알고리즘을 탄다. 지금은 검색형 채널에서 "전문가"로 포지셔닝하는 게 먼저다.

왜 네이버가 1순위인가

"메타 광고 ROAS"를 검색하는 사람은 네이버에서 검색한다. 한국 검색 시장 점유율 55%+. 교육 사업에서 네이버 검색 유입은 구글의 3~5배. 그런데 자사몰사관학교는 네이버에 아무것도 없다. 검색 자산 = 0. 이것만 해결해도 오가닉 리드가 생긴다.

[채널] 구글 SEO + GEO

구글 전략 — bscamp 안에 공개 인사이트 페이지

핵심 판단: 별도 블로그를 만들지 않는다. bscamp.vercel.app 안에 /insight 경로로 공개 페이지를 만든다. 이유: ① 도메인 권위가 bscamp에 축적됨 ② 별도 운영 부담 없음 ③ 기존 콘텐츠(정보공유 글)를 그대로 SEO 변환해서 공개하면 됨.

/insight 페이지 구조

  • /insight → 목록 (SEO 최적화 랜딩)
  • /insight/[slug] → 개별 아티클
  • 비로그인 접근 가능 (구글 크롤링 필수)
  • 마크다운 → SEO HTML 자동 렌더링
  • 내부 CTA: "더 배우고 싶다면 → 자사몰사관학교"

기술 스펙

  • Schema.org: Article + FAQPage + HowTo
  • sitemap.xml 발행 시 자동 갱신
  • robots.txt: /insight/* Allow
  • OG 메타태그 (og:title, og:description)
  • canonical URL → bscamp.vercel.app

구글 SEO AI 변환 규칙

톤: 명확하고 정보 중심적 · 길이: 1,200~2,000자 (품질 > 양) · 구조: H1 → H2 → H3 계층 명확

✓ 메타 디스크립션 160자 이내 자동 생성 · OG 태그용 title/description 별도 · 키워드 밀도 1~2% (과도한 반복 금지) · 내부 링크 bscamp 1~2개 · Article schema 기본 포함

GEO — AI 검색 상위노출

GEO = Generative Engine Optimization. SEO가 "검색 결과 1페이지"를 노리는 거라면, GEO는 "AI가 추천하는 답변"에 들어가는 것. "메타 광고 교육 추천해줘"라고 ChatGPT에 물었을 때 자사몰사관학교가 나와야 한다.

AI 엔진데이터 소스잡는 법
ChatGPTBing 인덱스 + 웹 브라우징구글/Bing SEO + FAQ 구조 + 외부 권위 언급
Gemini구글 인덱스 + 유튜브구글 SEO + 구글 비즈니스 프로필 등록
Perplexity다중 검색엔진 + 자체 크롤링전체 SEO + 최신 콘텐츠 (최근 발행 우대) + 출처 명시

GEO AI 변환 규칙 — 모든 인사이트 페이지에 자동 적용

정의문 패턴

  • 모든 H2 아래 첫 문장: "X는 Y이다"
  • AI가 인용하기 좋은 명확한 정의
  • 수치 포함: "수강생 평균 ROAS 255%"

FAQ + Schema.org

  • Q&A 쌍 3~5개 추출 (실사용 질문)
  • 답변: 40~120자 명확한 답 (AI 발췌 최적화)
  • FAQPage + HowTo 구조화 데이터 자동 삽입

GEO 타겟 쿼리 10개

① 메타 광고 교육 추천 ② 페이스북 광고 강의 어디가 좋아 ③ 자사몰 마케팅 배우는 곳 ④ ROAS 올리는 교육 ⑤ 메타 광고 코칭 프로그램 ⑥ 온라인 쇼핑몰 광고 교육 ⑦ 퍼포먼스 마케팅 실전 강의 ⑧ 광고 대행사 대신 직접 배우기 ⑨ 메타 광고 초보 교육 ⑩ 자사몰 매출 올리는 강의

6Newsletter + Hub

[채널] 뉴스레터 + 콘텐츠 허브

이미 가진 자산 — 활용만 하면 된다

뉴스레터 파이프라인은 이미 구축 완료. email_logs 기반 발송, 구독자 1,095명. 문제는 "보낼 콘텐츠"가 부족했던 것. 오가닉 채널 파이프라인이 돌면, 주 2회 생산되는 콘텐츠가 자동으로 뉴스레터 소재가 된다.

뉴스레터 전략

  • 주 1회 발행 (금요일, 콘텐츠 2건 요약)
  • 구조: 이번 주 인사이트 → 핵심 데이터 → CTA
  • 기존 email_logs + 발송 파이프라인 활용
  • 새 콘텐츠 발행 시 자동 연계 옵션
  • 뉴스레터 → 블로그 링크 → 체류시간 향상

콘텐츠 허브 = bscamp 자체

  • bscamp.vercel.app이 콘텐츠 허브 역할
  • /insight: 공개 아티클 (구글 SEO)
  • /admin/contents: 내부 정보공유 (원본)
  • KnowledgeService: RAG 기반 답변 생성
  • 벤치마크 대시보드: 데이터 시각화

콘텐츠 순환 구조 — 하나를 쓰면 다섯 곳에서 일한다

리드 마그넷 — 벤치마크 리포트 자동 생성

벤치마크 DB 7,366 rows를 리드 마그넷으로 전환. 매월 1일 자동 생성: "2026년 3월 메타 광고 업종별 벤치마크 리포트" — 업종별 평균 ROAS, CPA, CPM 요약 + 인사이트. 블로그/카페에 요약 게시 → 전문 리포트 다운로드 시 이메일 수집 → 뉴스레터 구독자 확대.

이것은 Phase 3 항목이지만, 전략적으로 가장 강력한 리드 마그넷. 아무도 이 수준의 데이터를 무료로 제공하지 않는다.

▶ 에이전트팀 개발 — 외부 의존성 없음, 바로 시작 가능

1-5 어드민 배포 UI 기초 에이전트팀 3시간 글 편집 페이지에 "오가닉 배포" 섹션 추가. 채널 체크박스 + "구글 SEO에 발행" 버튼. channel_distributions INSERT

✅ P1 완료 시: 어드민에서 글 작성 → "발행" 클릭 → 구글 인사이트 페이지 자동 생성 + GEO 마크업 자동 적용. Smith님 추가 작업 0.

📈 네이버 상위노출 로직

C-Rank + DIA + 키워드 배치 + 서식 최적화
C-Rank (신뢰도)
• 한 분야 전문성 유지 (자사몰/마케팅/광고)
• 총 포스팅 수, 이웃 수, 방문자 수 = 블로그 지수
• AI 글 감지 우회: 개인 경험 + 구어체 + 불규칙 문장

DIA (문서품질)
• 제목 30자 이내, 키워드 앞쪽 배치
• 본문 키워드 5~6회 자연 반복
• 이미지 직접 제작/촬영, alt 태그에 키워드
• 사진↔글 교차, 인용구 1~2, 구분선 2~3

티어별 글 구조
• Tier A 전환글: 1,800~2,500자 / 이미지 8~12장 / 비포↔애프터 + CTA
• Tier B 비교글: 1,500~2,000자 / 이미지 7~10장 / A vs B 표 + 추천
• Tier C 탐색글: 1,200~1,800자 / 이미지 5~8장 / 팁 3가지 + 시리즈 연결
• Tier D 정보글: 800~1,200자 / 이미지 3~5장 / 정의 + 예시 + 관련 링크

주간 발행 (주 3회)
월: Tier C/D 정보글 (블로그 지수) · 수: Tier B 비교글 (설득) · 금: Tier A 전환글 (CTA)
C-Rank DIA 4 Tier 글구조 주 3회 발행

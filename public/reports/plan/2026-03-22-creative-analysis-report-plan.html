수강생이 듣고 싶은 건 딱 하나

"내 광고가 왜 안 되는 건데? 어떻게 고치면 되는 건데?"

수강생은 히트맵이 예쁜지, 임베딩이 뭔지 관심 없다. 돈이 되는 광고를 만들고 싶을 뿐이다.

그래서 우리의 소재 분석은 총가치각도기 3축에서 시작하고, 3축으로 끝난다.

각 분석 섹션이 3축 어디에 대응하는가

리포트의 모든 섹션은 "이 소재의 어떤 점이 기반/참여/전환에 영향을 주는가"를 설명하기 위해 존재한다.

분석 섹션🟢 기반 (보긴 하나)🟡 참여 (관심 갖나)🔴 전환 (돈 되나)
시선 분석 (DeepGaze)핵심 — 첫 시선이 가는 곳 = 훅의 힘. 3초 안에 시선을 잡는가?시선이 머무는 곳 = 공감 포인트. 감정을 유발하는 영역이 주목받는가?CTA에 시선이 가는가? CTA 주목도 = 클릭 확률의 선행 지표.
5축: Visual핵심 — 훅 유형, 색상 대비, 구성 = 스크롤 멈추게 하는 힘인물/얼굴 노출 = 공감 유도제품 가시성 = 뭘 파는지 인지
5축: Text헤드라인 = 첫 인상핵심 — 핵심 메시지, 소셜프루프 = "나도 그래!" 공감CTA 텍스트 = 행동 유도
5축: Psychology호기심 트리거 = 멈춤핵심 — 감정(신뢰/공포/기쁨) = 반응+공유 유도긴급성 + 오퍼 = 지금 사야 하는 이유
5축: Quality제작 품질 = 신뢰 첫인상가독성 = 메시지 전달력핵심 — 피로도 = 같은 소재 반복 시 전환율 하락
임베딩 유사도--핵심 — 유사 소재 중 성과 좋은 것 = "이런 변형이 더 팔린다"
소재↔LP 일관성--핵심 — 소재와 LP가 불일치하면 이탈. 일관성 = 전환의 전제조건.
영상: Audio나레이션 첫 문장 = 청각 훅BGM 감정 = 분위기 + 공감엔딩 나레이션 = CTA 강화
영상: Structure핵심 — 훅 구간(0~3초) 구조 = 이탈 방지장면 수/페이싱 = 지루함 방지엔딩 CTA 유형 = 전환 마무리
핵심 원칙: 모든 분석 결과를 "그래서 기반/참여/전환 중 뭐에 영향을 주는데?"로 해석해야 한다.
히트맵이 예뻐도 기반 점수가 낮으면 의미 없다. 5축이 완벽해도 LP 일관성이 낮으면 전환이 안 된다.
→ 리포트의 마지막은 항상 "3축 중 어디가 약하고, 뭘 고치면 되는지"로 끝난다.

리포트가 수강생에게 전달하는 가치

기반이 약할 때

"사람들이 네 광고를 안 봐."

→ 시선 분석: 훅 영역 주목도 낮음
→ Visual: 색상 대비 low, 훅 유형 weak
→ 영상: 첫 3초 구조 문제

해결: "벤치마크 상위 광고는 이런 훅을 쓴다. 네 것과 비교해봐."

참여가 약할 때

"보긴 하는데 반응이 없어."

→ Psychology: 감정 유발 없음 (none)
→ Text: 소셜프루프 없음, 질문형 아님
→ 시선: 공감 포인트에 시선 안 감

해결: "이 업종에서 후기 추가 시 참여 +44%. 실제 후기 넣어봐."

전환이 약할 때

"관심은 있는데 안 사."

→ LP 일관성: CTA 불일치 (45/100)
→ 시선: CTA 주목도 낮음
→ Quality: 소재 피로도 high
→ Psychology: 긴급성 없음

해결: "소재-LP CTA 통일 + 긴급성 추가 + 소재 변형 필요."

분석 흐름: 데이터 → 해석 → 처방

업계 서비스들은 대부분 데이터 단계에서 멈춘다.
Neurons: "여기에 시선이 간다" → 끝.
Motion: "이 소재가 CTR 높다" → 끝.

우리는 데이터 → 해석(왜?) → 처방(어떻게?)까지 간다.
"여기에 시선이 간다" → "왜? 색상 대비가 높으니까" → "그래서? 기반은 좋은데 CTA에 시선이 안 가니까 전환이 약해. CTA 색상을 바꿔."

3계층 분석 모델

계층무엇도구수강생이 보는 것
Layer 1: 데이터소재에서 뭘 감지했는가DeepGaze(시선) + Gemini(5축) + Embedding(유사도)히트맵, 속성 태그, 유사 소재
Layer 2: 해석왜 그런 결과인가3축 매핑 + 벤치마크 비교 + LP 일관성"기반 약함 — 훅 주목도 32%, 벤치마크 평균 67%"
Layer 3: 처방어떻게 고칠 것인가순환 학습 + 벤치마크 콘텐츠 + 요소별 Lift"후기 추가(+44%) > CTA 색상 변경(+12%) > 긴급성 추가(+8%)"

🖼️ 이미지 소재 분석 — 정의서

흐름: 총가치 스코어카드(어디가 약한지) → 시선(왜 약한지) → 5축(구체적 원인) → 유사도(다른 소재와 비교) → LP일관성(전환 단절 체크) → 처방(어떻게 고칠지)

섹션 1: 총가치각도기 스코어카드

수강생이 보는 것: "내 소재는 기반 67점(상위 23%), 참여 45점(상위 55%), 전환 32점(상위 78%)이다."

🟢 기반 점수 67/100

벤치마크 상위 23%
CTR 2.43% (벤치마크 2.21%)
3초시청률: 이미지라 N/A

🟡 참여 점수 45/100

벤치마크 상위 55%
참여: 12/만 (벤치마크 27/만)
댓글+공유 부족

🔴 전환 점수 32/100

벤치마크 상위 78%
ROAS: 0 (구매 미발생)
LP 일관성: 64/100

→ 종합 Impact Score: 48/100 (3축 가중 평균. 가중치: 기반 30% / 참여 30% / 전환 40%)

→ "참여와 전환이 약하다. 아래에서 왜 약한지 본다."

섹션 2: 시선 분석 — 👁 어디를 보는가

수강생이 보는 것: "사람들 시선이 텍스트에만 가고 제품이랑 CTA는 안 봐."

🔥 원본 + 히트맵 (나란히)

좌: 원본 이미지
우: DeepGaze 히트맵 (warm-to-cold)
+ 시선 포인트 순서 번호 (1→2→3→4→5)

👁 Gazeplot + Focus Map

Gazeplot: 시선 이동 경로 (선으로 연결)
Focus Map: 인지 영역(녹) / 미인지(회)
→ "CTA가 미인지 영역에 있다 = 문제"

AOI 분석: CTA 주목도 12% (벤치마크 평균 34%) / 제품 주목도 45% / 텍스트 주목도 78%

인지 부하: low → "단순한 구성이라 보기 편하다. 하지만 CTA가 묻힌다."

섹션 3: 5축 디테일 — 🧠 왜 그런가

수강생이 보는 것: "어떤 요소가 성과에 영향을 주는지"

🎨 Visual

스타일: professional / 훅: benefit / 구성: center
색상: warm톤, high대비 / 팔레트: 🟤🟫⬜
인물: 얼굴 없음 ← ⚠ 시선 집중 감소 요인
제품 가시성: high / 텍스트 비율: 20%

📝 Text

핵심: "사이즈 불만족 시 100% 환불"
헤드라인: benefit형 / CTA: 없음 ← ⚠ 클릭 유도 부재
오버레이: ["100% 환불", "빅사이즈"]
소셜프루프: 후기 없음 ← ⚠ 참여 약화 요인

🧠 Psychology

감정: trust(신뢰) / 긴급성: none ← ⚠ 전환 약화
트리거: none / 오퍼: 환불 보장
사회적 증거: none ← ⚠ 참여 약화

🎯 Quality + Attention

제작 품질: professional ✅
가독성: high ✅
인지 부하: low ✅
CTA 주목도: 12% ← ⚠ 매우 낮음
피로도: null (유사 소재 비교 필요)

섹션 4: 임베딩 & 유사도 — 🔗 비슷한 소재와 비교

수강생이 보는 것: "내 소재와 비슷한데 성과 좋은 소재가 있다"

  • 같은 계정 유사 소재 Top 3 — 썸네일 + 코사인 유사도 + 성과 차이
  • 벤치마크 유사 소재 Top 3 — "이 소재와 비슷한데 CTR이 2배인 소재. 차이: 얼굴 노출 + 후기 추가."
  • 피로도 위험 — 유사 소재 비율 높으면 "오디언스가 같은 소재에 지쳤다. 다른 각도 필요."

섹션 5: 소재↔LP 일관성 — 🔗 끊기는 곳 찾기

수강생이 보는 것: "소재에서 약속한 걸 LP에서 지키고 있나?"

  • 종합 일관성: 64/100
  • 메시지 78 ✅ / CTA 45 ❌ / 오퍼 50 ⚠️
  • 불일치: "소재는 '100% 환불' 강조하는데, LP 상단에 환불 정책 없음 → 이탈 유발"

섹션 6: AI 개선 제안 — 💡 뭘 고칠 것인가 (처방)

수강생이 보는 것: "우선순위대로 이걸 고치면 된다"

Impact 순 정렬 (높은 것부터):

1. ⭐ 실제 후기 추가 → 참여 예상 +44% (이 업종 순환학습 데이터)
2. 🎯 CTA 텍스트 명시 ("지금 구매하기") → 전환 개선 (CTA 주목도 12%→34% 목표)
3. ⚡ 긴급성 추가 (한정 수량/기간) → 전환 예상 +8%
4. 🔗 LP 환불 정책 상단 배치 → 소재↔LP 일관성 64→80+ 목표
5. 👤 얼굴 노출 이미지 테스트 → 기반(시선 집중) 예상 +40%

🎬 영상 소재 분석 — 이미지 6섹션 + 영상 전용 4섹션

이미지 분석의 6개 섹션을 그대로 포함하고, 영상만의 시간축 분석 4개 섹션이 추가된다.
영상의 핵심: 시간에 따른 변화. 이미지는 한 장이지만 영상은 매 초가 다르다.

↑ 섹션 1~6: 이미지와 동일

총가치 스코어카드 / 시선(히트맵은 구간별) / 5축(Audio+Structure 추가) / 임베딩 / LP일관성 / 개선제안

섹션 7: 장면 타임라인 — ⏱️ 어떤 구조로 흘러가는가

수강생이 보는 것: "내 영상이 어떤 흐름으로 구성됐고, 각 구간이 제 역할을 하는가"

구간 유형 역할 (3축) 설명 ─────────────────────────────────────────────────────────────────────── 0~3초 [훅] 🟢 기반 — 스크롤 멈추기 화장 무너지는 피부 클로즈업으로 공감 유도 4~12초 [데모] 🟢🟡 기반+참여 제품 소개 + 전직 승무원 신뢰도 부여 13~22초 [증거] 🟡 참여 — 공감+신뢰 사용 전후 비교 + 성분 설명 23~31초 [CTA] 🔴 전환 — 행동 유도 구매 유도 + 할인 오퍼
  • 구간별 시선 히트맵 — 각 구간의 키프레임에 DeepGaze 히트맵 오버레이
  • 시선 타임라인 그래프 — X축: 시간, Y축: 주목 영역 변화 (제품/얼굴/텍스트/CTA)
  • 각 구간이 3축 중 어디에 대응하는지 표시 → "훅 구간이 약하면 기반이 무너진다"

섹션 8: 오디오 분석 — 🎙 귀로 듣는 것도 분석

수강생이 보는 것: "나레이션이 화면과 맞는지, BGM이 적절한지"

오디오 메타

유형: mixed (나레이션+BGM)
BGM: pop / 감정: upbeat
효과음: sparkle, swoosh
→ 🟡 참여: BGM 감정이 제품 무드와 맞는가?

나레이션 전문 + 시각 일치

"어떤 파데를 써도 피부 화장이 금방 무너진다면 딱 이거 하나만 바꿔봐..."

→ 🟢 기반: 첫 문장이 청각 훅 역할 하는가?
→ 나레이션↔화면 일치도: 말하는 내용이 보이는 것과 맞는가?

섹션 9: 영상 시선 Canvas — 👁 재생하면서 시선 따라가기

인터랙티브 뷰 (bscamp 앱 전용 — 모찌리포트에서는 구간별 키프레임으로 대체)

HTML5 video 재생 + canvas 히트맵 오버레이 → 시선이 실시간으로 따라감
구간 색상: ●빨강(훅 0~3초) → ●파랑(제품 3~8초) → ●초록(CTA 8초+)

video-heatmap-overlay.tsx: video timeupdate → 가장 가까운 프레임 fixations → canvas 가우시안 히트맵

수강생이 보는 것: "내 영상을 재생하면 사람들 시선이 어디로 가는지 실시간으로 보인다."

섹션 10: 구조 분석 — 🎬 영상의 뼈대

수강생이 보는 것: "내 영상 구조가 최적인가, 뭘 바꿔야 하는가"

  • 페이싱: fast (빠른 전환 → 지루함 방지 ✅ but 정보 과부하 가능)
  • 장면 수: 4개 / 평균 장면 길이: 7.8초
  • 훅 유형: 문제제기 ("화장이 무너진다면") → 🟢 기반 강화
  • 엔딩 CTA: 할인 오퍼 ("지금 구매") → 🔴 전환 강화
구조 최적화 제안:
• 훅 구간 3초 → OK (벤치마크 평균 2~3초 이내)
• 증거 구간에 실제 후기 추가하면 참여 강화
• 엔딩 CTA 전 0.5초 정지 프레임 추가 → CTA 주목도 향상

🏆 bscamp만의 차별점 — 어디에도 없는 것

시선 분석만 하는 Neurons, 성과만 보여주는 Motion, 요소만 태깅하는 Vidmob.
우리는 "소재 → 시선 → 해석 → LP까지 → 전환까지 → 처방까지" 풀 퍼널을 커버한다.
1. 총가치각도기 연동

모든 분석이 기반/참여/전환 퍼널에 매핑된다. "히트맵이 예쁘다"가 아니라 "기반 점수가 67점이다."로 읽힌다.

다른 곳: 시선 점수와 비즈니스 성과가 분리됨.

2. 소재↔LP 일관성

소재만 분석하는 게 아니라 LP까지 이어지는 흐름을 본다. 메시지/시각/CTA/오퍼 4축. 끊기는 곳 = 이탈 지점.

다른 곳: LP는 아예 안 봄.

3. 순환 학습 기반 처방

"후기 추가 = +44%"는 추측이 아니라 실제 데이터. 소재 변경 → 성과 변화를 추적해서 축적한 실증 데이터.

다른 곳: 일반적 베스트 프랙티스 권고.

4. 벤치마크 콘텐츠 비교

숫자 비교("네 CTR이 낮아")가 아닌 소재 자체 비교("잘 되는 광고는 이런 훅을 쓴다"). 임베딩으로 유사 소재 매칭.

다른 곳: 숫자 벤치마크만.

5. 영상 시간축 분석

장면 타임라인 + 구간별 히트맵 + 오디오 분석 + Canvas 오버레이. 영상을 초 단위로 해부.

다른 곳: Brainsight만 영상 히트맵, 나머지는 이미지만.

6. 3계층 분석

데이터(뭘 감지) → 해석(왜) → 처방(어떻게). 대부분 서비스는 Layer 1에서 멈춤. 우리는 Layer 3까지.

다른 곳: "여기에 시선이 간다" → 끝.

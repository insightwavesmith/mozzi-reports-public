📊 분석 프레임워크 — 고객 여정 4단계

"모든 숫자는 고객이 만든다. 보고, 듣고, 생각하고, 행동한 것의 합."
단계고객 행동핵심 메트릭분석 대상
1. 감각👁👂 보고 + 듣고3초시청률, thumbstop훅, 첫 프레임, 시선
2. 사고🧠 생각하고완시청률, hold rate씬 구조, 감정, 설득
3. 행동(선행)🖱 클릭CTR, CPCCTA, 긴급성, 오퍼
4. 행동(후행)💳 구매전환율, ROASLP 일관성, 오퍼 구체성

처방 규칙

• 3초시청률 < 벤치마크 → 위 5개 중 현재 안 쓰는 tactic 권장
• 시선이 텍스트에만 → 비주얼 훅 강화 (인물 표정, 극적 비주얼)
• 첫 프레임이 광고 느낌 → 오가닉 피드 블렌딩 (Andromeda)

Andromeda 훅 원칙

• 첫 프레임이 유저 오가닉 피드와 시각적으로 유사해야 점수 높음
• 프로덕션 퀄리티 너무 높으면 광고 인식 → 스킵률 증가
• UGC 스타일 첫 프레임이 Andromeda 친화적

🎬 씬 구조 — Hold Attention

레퍼런스 데이터

• 완시청률 높은 영상: 5초 간격 씬 전환 (Motion)
• 감정 4단계 > 3단계: 공감→충격→기대→행동 (참여율 높음)
• 제품 시연(application) 있는 영상: 전환율 높음 (Vidmob 뷰티)
• 임상/수치: 신뢰도↑, 단 텍스트 시선 독점 주의

처방 규칙

• 시선 텍스트 독점 → "비주얼로 승부할 구간에서 자막 줄이기"
• 제품 시선 < 10% → "제품 풀스크린 2-3초 삽입"
• 씬 전환 없이 5초+ → "씬 전환 포인트 추가"
• 감정 전환 < 2회 → "충격/서프라이즈 포인트 추가"

🖱 CTA — Get People to Site

절대 규칙: "CTA 버튼을 영상에 넣어라" ← ❌ 금지

영상에서 해야 할 것

✅ "지금 눌러야 하는 이유" 만들기
• 긴급성: "오늘까지" / "재고 XX개" / 타이머
• 혜택 구체화: "40% → 19,800원, 이번 주까지"
• 나레이션 행동 유도: "아래 링크 눌러봐"
• 시각적 유도: 아래 방향 화살표, 스와이프 제스처

처방 규칙

• urgency = none → 긴급성 추가
• 할인 구체성 없음 → 가격+기한 명시
• CTA가 마지막에만 → 15초쯤 중간 CTA 자막
• 나레이션 행동 유도 없음 → 멘트 추가

📝 텍스트/자막

• 위치 세이프티존 밖 → "중앙으로 이동"

🎨 비주얼 스타일

레퍼런스 데이터

• UGC > professional: 참여율 높음 (Vidmob)
• 얼굴 노출: 시선 집중 +40% (Neurons)
• 제품 사용 장면: 전환율 높음 (Vidmob 뷰티)
• 4:5 세로 > 1:1 정사각형: 성과 +15% (Billo)

처방 규칙

• professional + 참여 낮음 → "UGC 스타일 테스트"
• 얼굴 미노출 → "인물 노출 추가 (+40%)"
• 제품 시선 낮음 → "제품 클로즈업 씬 추가"

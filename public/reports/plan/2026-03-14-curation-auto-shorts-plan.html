🚀
트리거: 정보공유 버튼 클릭
큐레이션 콘텐츠 (제목 + 요약 + URL + 원문) 가져오기 → 기존 블로그 포스팅 실행 + 숏츠 파이프라인 병렬 실행
1
스크립트 생성 — GPT-4o
큐레이션 콘텐츠를 60초 숏츠 대본으로 변환
구성: 훅(10초) + 핵심 내용(40초) + CTA(10초) | 한국어 자연스러운 나레이션 스타일
2
나레이션 생성 — ElevenLabs TTS
한국어 AI 음성으로 나레이션 오디오 생성
자연스러운 억양·속도 조정 / 선택: 여성/남성 목소리
3
비주얼 소재 확보
옵션 A (빠름): 텍스트 오버레이 슬라이드 (Creatomate 템플릿)
옵션 B (퀄리티): Pexels API로 관련 스톡 영상 자동 검색
옵션 C (브랜딩): Replicate/Leonardo AI로 키 이미지 생성
→ MVP는 옵션 A로 시작, 이후 업그레이드
4
영상 조립 + 자막 — Creatomate API
9:16 세로형 영상으로 조립 / 자동 자막 삽입 (눈에 띄는 스타일)
출력: MP4 1080×1920, 30~60초
5
메타데이터 생성 — GPT-4o
YouTube 업로드용 제목 / 설명 / 해시태그 자동 생성
숏츠 알고리즘 최적화 제목 (훅형) / #해시태그 3~5개
6
YouTube 업로드 — YouTube Data API v3
자동 업로드 → #Shorts 해시태그로 숏츠 지정
업로드 완료 후 URL 반환 → 큐레이션 대시보드에 표시

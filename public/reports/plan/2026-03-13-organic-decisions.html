오가닉 콘텐츠를 RAG 지식베이스에 넣을 것인가?
P0 필수 — 개발 착수 전 결정
외부 발행용 블로그/인스타 콘텐츠가 Q&A 챗봇의 지식에 포함되어야 하는가?
옵션설명장점리스크
A. 포함 오가닉 콘텐츠도 임베딩 → knowledge_chunks에 저장 QA 답변 소스 풍부해짐, RAG 정확도 향상 외부 마케팅 톤이 QA 답변에 혼입 가능
B. 제외 오가닉은 임베딩 안 함, 외부 발행 전용 내부 교육 톤 보호, 깔끔한 분리 동일 주제 콘텐츠가 QA에서 활용 안 됨
C. 선택적 관리자가 콘텐츠별로 "RAG 포함" 체크 유연함, 필요한 것만 투입 관리 부담, 매번 판단 필요
모찌 추천: B (제외) — 오가닉은 마케팅 톤이라 QA와 섞이면 답변 톤이 뒤죽박죽될 수 있음. 나중에 필요하면 C로 전환하기 쉬움.
2
외부 발행 톤 가이드
P0 필수 — AI 초안 품질에 직결
오가닉 콘텐츠의 말투/톤을 어떻게 설정할 것인가?

현재 style-learner는 Smith님의 Q&A 답변 말투를 학습해서 "교육 코칭 톤"으로 AI 답변을 생성. 오가닉은 이 톤과 달라야 함.

옵션톤예시
A. 전문가 업계 전문가 포지셔닝, 깔끔한 존댓말 "상세페이지 전환율을 높이는 5가지 핵심 요소를 정리했습니다."
B. 친근 전문가 전문성 + 친근한 반말/혼용 "상세페이지 전환율 올리는 비결, 5가지만 기억하세요!"
C. 코칭 연장 현재 style-learner 톤 그대로 "이 부분은 제가 실제로 수강생분들에게도 강조하는 건데요~"
모찌 추천: B (친근 전문가) — 네이버 블로그 SEO에서 친근한 톤이 체류시간 높임. 동시에 전문성으로 신뢰 확보.

Phase 1 지원 채널 범위
P0 필수 — 개발 범위 결정
Phase 1 (MVP)에서 어떤 채널부터 지원할 것인가?
옵션채널이유기간
A. 블로그만 네이버 블로그 API 발급 완료, SmartEditor 리서치 완료, 키워드 스크립트 완성 ~3일
B. 블로그+카페 네이버 블로그 + 카페 같은 네이버 OAuth, 콘텐츠 포맷 동일, 크로스포스팅 쉬움 ~4일
C. 블로그+인스타 네이버 블로그 + 인스타 텍스트+영상 두 축 동시 확보, Meta API 이미 익숙 ~5일
D. 전부 블로그+카페+유튜브+인스타+틱톡 한번에 다 하기 ~2주+
모찌 추천: A (블로그만) — 모든 인프라 준비 완료. 빠르게 MVP 나오고 실데이터 쌓은 후 확장.

콘텐츠 승인 프로세스
P1 중요
AI가 생성한 초안을 어디서 어떻게 검수하고 발행할 것인가?
옵션프로세스장점단점
A. 수동 검수 AI 초안 → Smith님 bscamp에서 확인 → 승인 버튼 → 자동 발행 품질 보장, 실수 방지 Smith님 시간 소요
B. 자동 발행 AI 초안 → SEO 점수 80점 이상이면 자동 발행 (슬랙 알림만) 완전 자동, 운영 부담 0 AI 품질 리스크, 초기에 위험
C. 단계적 초기 A → SEO 점수 신뢰 쌓이면 → B로 전환 안전하면서 점진적 자동화 전환 시점 판단 필요
모찌 추천: C (단계적) — 처음 10~20개는 수동 검수로 품질 기준 잡고, 이후 자동 전환.

이미지 소싱 방법
P1 중요
블로그 글에 들어가는 이미지 7~10장을 어떻게 확보할 것인가?
옵션방법비용품질자동화
A. Unsplash/Pexels 무료 스톡 이미지 API 무료 보통 (범용적) 완전 자동
B. AI 생성 DALL-E / Midjourney로 맞춤 이미지 건당 $0.04~0.08 높음 (맞춤) 완전 자동
C. 기존 자산 강의 자료, 대시보드 캡처 등 내부 이미지 무료 높음 (오리지널) 수동
D. 혼합 핵심 이미지 = C(내부), 보조 이미지 = A(스톡) 무료 높음 반자동
모찌 추천: D (혼합) — 핵심 이미지(대시보드 캡처, 성과 그래프)는 오리지널로, 나머지는 Pexels API로 자동 매칭.
참고: 네이버 SEO에서 유니크 이미지가 중요. 스톡만 쓰면 C-Rank 불이익 가능. 최소 3~4장은 오리지널 권장.

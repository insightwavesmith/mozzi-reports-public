👤 2. 고객 페르소나 (엠타트업 마케팅축)

엠타트업 마케팅축: "어떤 욕구가 있고, 어떤 대안으로, 접근시키면 → 돈을 벌겠구나"

🌱 페르소나 A: 초기 자사몰 대표

매출 1억↓광고비 100만↓
욕구"내 브랜드 자사몰에서 매출을 만들고 싶다"
고통광고비 쓸 돈도 없고, 어디서부터 시작해야 할지 모름
대안스마트스토어만 하다가 자사몰 필요성은 알지만 방법 모름
접근체험단, SNS 마케팅, 무료/저비용 방법부터
실제 QA체험단 지원자 0명, 광고비 소진 안 됨, 메타 Verified 오류

🚀 페르소나 B: 성장기 대표 ← 핵심 타겟

매출 1~10억광고비 100~500만
욕구"광고비 대비 효율(ROAS)을 올려서 이익을 내고 싶다"
고통광고비는 나가는데 전환이 안 됨, 데이터 읽는 법 모름
대안대행사 맡기다가 직접 해보려는데 복잡함
접근메타 광고 최적화, 상세페이지 개선, 픽셀/전환 추적
실제 QA상품별 데이터 분산, 비밀링크 SKU 통합, 머신러닝 상태

🏢 페르소나 C: 확장기 대표

매출 10억+광고비 500만+
욕구"시스템화해서 스케일업하고 싶다"
고통광고 의존도 높음, 인하우스 역량 부족
대안대행사+인하우스 하이브리드, 자동화
접근카탈로그 광고, ASC+, 경매 최적화, 배수효과
실제 QA메타 광고와 공구 채널 병행, 경매 입찰 최적화

🔍 3. 키워드 5단계 스펙트럼

엠타트업 핵심: Level 3~4 구간 조작이 전환을 만든다 — 여기서 물길을 만들어야 함
L1 순수정보 — ROAS 뜻, CTR 뜻, 마케팅이란
L2 관심탐색 — 메타광고 효과, 상세페이지 중요성
L3 비교검토 — 대행사 vs 인하우스, 자사몰 vs 스마트스토어
L4 구매직전 — 상세페이지 제작, 메타광고 세팅, 체험단 모집
L5 전환 — 자사몰사관학교 후기, 마케팅 교육 추천

💬 7. QA 기반 콘텐츠 아이디어

수강생이 실제로 물어본 것 = 잠재 고객도 궁금한 것

❓
체험단 모집했는데 지원자가 0명이에요
→ 키워드: "체험단 모집 방법 2026" L4
❓
메타 광고 머신러닝이 끝나지 않아요
→ 키워드: "메타광고 머신러닝 기간 해결" L4
❓
Meta Verified 시작하기 버튼이 안 눌려요
→ 키워드: "메타 인증 오류 해결방법" L4
❓
비밀링크로 랜딩하면 데이터가 분산돼요
→ 키워드: "메타 픽셀 데이터 통합 방법" L3
❓
메타 광고와 인스타 공구 같이 해도 되나요?
→ 키워드: "인스타 공구 메타광고 병행" L3

🔗 연동 설계: 소재 요소 × 3단계 벤치마크

핵심 아이디어

소재의 각 요소(hook_type, style, cta_type 등)별로 ROAS만이 아닌 기반·참여·전환 3단계 지표를 계산.
→ "이 hook 유형은 시청률은 좋지만 전환이 약하다" 같은 단계별 진단이 가능.

가중평균 계산 방식

예: hook_type = "problem" + format = "video"의 3초시청률 벤치마크

1. creative_element_analysis에서 hook_type='problem' AND format='video'인 ad_id 추출
2. daily_ad_insights에서 해당 ad_id의 video_p3s_rate × impressions 합산
3. 가중평균 = SUM(video_p3s_rate × impressions) / SUM(impressions)
4. 최소 광고비 10,000원 이상만 포함

⚠️ hook_type "none" 처리

현재

L1 태깅에서 hook을 인식 못하면 "none"
벤치마크에 "none"이 포함됨
동영상인데 hook 없음 = 비정상

개선

1. 동영상 "none" → L1 재태깅 (프롬프트 강화)
2. 이미지 "none" → "direct" (바로 제품 노출)로 rename
3. 벤치마크에서 동영상 none은 제외 (샘플 부족)
